# مداد العلم – منصة تعليمية متكاملة

مداد العلم هي منصة SaaS حديثة تتيح لأي مدرس أو أكاديمية إنشاء منصة تعليمية إلكترونية احترافية خلال دقائق.

## المميزات

- **تمرير سلس** باستخدام Lenis
- **حركات احترافية** باستخدام GSAP + ScrollTrigger
- **تأثيرات ظهور** باستخدام AOS
- **أداء محسن** مع GPU Acceleration
- **تصميم عصري** مع Tailwind CSS

## التقنيات المستخدمة

- **Frontend**: HTML5, Tailwind CSS, JavaScript (ES6)
- **Backend**: Firebase (Authentication, Firestore, Storage)
- **Animations**: Lenis, GSAP, AOS

- **الأدوات**: Font Awesome 6, Google Fonts
